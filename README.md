 🛒 Supermarket Sales Dashboard | Power BI

  📌 Project Overview

This project is an interactive Power BI dashboard built using the Supermarket Sales Dataset. It provides insights into sales performance, customer behavior, product categories, payment preferences, and operational trends through interactive visualizations and dynamic filtering.

The dashboard also includes an Executive Insights page that automatically updates summaries and key business insights based on user selections.

---

 🎯 Business Objective

The objective of this dashboard is to help business managers and decision-makers monitor sales performance, identify customer trends, and support data-driven decisions using interactive analytics.

---

 📊 Dashboard Features

 Main Dashboard

* Monthly Revenue Trend
* Total Revenue
* Total Cost
* Total Profit
* Total Quantity Sold
* Total Transactions
* Average Customer Rating
* Revenue by Gender
* Revenue by Day of Week
* Revenue by Payment Type
* Revenue by Shift
* Interactive slicers for:

  * City
  * Product Line
  * Customer Type

 Executive Insights Page

* Dynamic selected filters
* Executive performance summary
* Highest Revenue Day
* Highest Revenue Shift
* Preferred Payment Method
* Dashboard information
* User guide
* Cross-page synchronized slicers

---

 📂 Dataset Information

* Dataset: Supermarket Sales Dataset
* Records: 1000
* Columns: 17
* Period: January – March 2019
* Cities: Yangon, Naypyitaw, Mandalay

 📈 Key Business Insights

* Revenue trends can be analyzed across different months.
* Customer purchasing behaviour varies across product categories.
* Revenue differs by weekday and operational shift.
* Payment preferences can be compared across customer segments.
* Interactive filters allow detailed city-wise, product-wise, and customer-type analysis.
* Executive Insights page dynamically updates based on report filters.
